HOW TO: Compute Plans

Compute Plan

To compute a new plan, you have to click the New Plan button in the navigation bar.

This opens a new view on the right. Here you can select the goals for the plan. In the top you can see the global goals. These goals have to be satisfied by all plans and are automatically included.

You start the plan computation with the button in the bottom right corner. To compute the plan can take some time.

When the computation finished, a new plan will appear in the navigation bar. The number in the orange circle indicates the utility of the plan.

Plan Information

For a selected plan the middle column provides you the following information:

info

The goals you selected.

 info

The properties which are additionally satisfied by the plan.

 info

The overall utility of the plan.

 info

The properties which are not satisfied by the plan.

If it is not possible to find a plan for the selected goals, the tool will tell you, that the selection of goals is unsolvable.

Plan Animation

In the left column you can play an animation of the plan.

To start the animation click the play button. With the arrow on the left an right of the play button you can can play on step of the plan at the time. With the right most button you can reset the animation.

Below the animation you see the current fuel/energy level of the components.

HOW TO: Ask Questions

Ask Question

To ask a question you can click the Ask Question button. This opens the question view.

You can ask questions of the structure: "Why not X?". Where X is a property, which is currently not satisfied by the plan.

By pressing the button in the bottom you start the computation of the answer.

Answers

There are different possible answers:

question_answer

If there is no reason, why the property is not satisfied, the system then answers you: "There exists a plan which satisfies the properties in your question."

This means you can add the property in your question to the goals to increase the utility.

question_answer

If is not possible to satisfy the property in your question, the system answers: "There is no plan which satisfies this properties. All global goals and the properties in the question can not be satisfied together."

This means you can not satisfy the property in your questions given the fuel, energy or time constraints.

question_answer

The property in your questions can be satisfied, if you give up at least you of the selected goals. The system then tells you which goals you have to give up.

No Plan Possible

If the system can not find a plan for the selected goals you can ask why? The system answers your question by showing you which goals can not be achieved together.

HOW TO: Finish

If you think you found the best plan (highest utility) possible you can leave the task with the Finish button.
