Task Introduction

In the following task you have to deliver packages from a start to destination location. To do so, you have 2 trucks which can perform the following actions:

drive: Drive between to locations which are connected with a road. load: Load one package if truck and package are at the same locations. unload: Unload a package at the current location of teh truck.

Both trucks have only a limited amount of fuel.
